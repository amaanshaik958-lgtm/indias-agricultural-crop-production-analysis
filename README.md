# indias-agricultural-crop-production-analysis
# 🌾 Comprehensive Analysis of Agricultural Production in India (1997–2021)

An interactive data visualization dashboard developed using **Tableau** to analyze India's agricultural production from **1997 to 2021**. This project helps users explore crop production trends, cultivated area, seasonal distribution, and state-wise agricultural performance through interactive visualizations.

---

## 📖 Project Overview

Agriculture is one of the major contributors to India's economy. Understanding agricultural production trends is important for farmers, researchers, policymakers, and students. This project transforms raw agricultural data into meaningful insights using Tableau dashboards.

The dashboard provides visual analysis of agricultural production across different crops, states, seasons, and years.

---

## ✨ Features

- 📊 Total Agricultural Production
- 🌱 Total Yield Analysis
- 📈 Year-wise Agricultural Production Trends (1997–2021)
- 🌾 Crop-wise Cultivated Area Comparison
- 🗺️ State-wise Agricultural Production Map
- 📉 State-wise Area Comparison
- 🍂 Season-wise Agricultural Production Share
- 🏆 Leading Agricultural Producing States
- 🔍 Interactive Filters (Crop, State, Season, Year)
